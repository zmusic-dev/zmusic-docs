## 简介

ZMusic Docs 是 ZMusic 的官方文档站点，当前站点基于 VitePress 构建，提供简体中文、English、繁體中文和日本語文档。

## 反馈

- 提交 [Issues](https://github.com/zmusic-dev/zmusic-docs/issues)
- 发起 [Pull Requests](https://github.com/zmusic-dev/zmusic-docs/pulls)

## 开发

```shell
git clone https://github.com/zmusic-dev/zmusic-docs
cd zmusic-docs
pnpm install
pnpm docs:dev
```

文档源码位于 `docs/`，VitePress 配置位于 `docs/.vitepress/`。

## 构建

```shell
pnpm docs:build
pnpm docs:preview
```
